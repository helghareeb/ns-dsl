\begin{letter}{Dr Devipriya Arumugam, Assistant Editor\\\textit{Scientific Reports}}
\opening{Dr Arumugam,}

Thank you for your continued handling of our manuscript, ``A decentralized neutrosophic decision
layer for global-state freshness in microservices architectures'' (Submission ID
c5ae1f20-9e89-4562-b69d-5dade5aca9d7). We have made the requested change to the data and code
availability and respond point by point below. No scientific content was altered.

\textbf{Editor comment.} The Zenodo link (DOI: 10.5281/zenodo.20932989) was not accessible. Please
provide the accessible link and the publicly available repository link so that the data can be
accessed, and provide the Data Availability statement both in the manuscript and in the submission
system.

\textbf{Author response.} We have removed all access restrictions. Both the Zenodo record and the
source-code repository are now \textbf{openly and publicly accessible}, with no token or private link
required:

\begin{itemize}
\item \textbf{Zenodo record (open):} \url{https://doi.org/10.5281/zenodo.20932989} --- the complete
raw per-trial outputs, the aggregated result summaries, the figures, the content-hashed calibration,
and the full implementation. The files now download directly, without any token (data under CC BY
4.0, code under the MIT licence).
\item \textbf{Public GitHub repository:} \url{https://github.com/helghareeb/ns-dsl} --- the live
source repository (implementation, simulator, ANTLR-based DSL and rules engine, baselines, and the
figure/table generators), under the MIT licence and cross-linked from the Zenodo record.
\item The aggregated result summaries underlying every figure and table also remain available with
the article as \textbf{Supplementary Information}.
\item We have updated the Data Availability and Code Availability statements in the manuscript to
reflect open access, and we have set the Data Availability declaration in the submission system to
the identical statement below.
\end{itemize}

\medskip
\textbf{Data Availability statement (identical in the manuscript and the submission system):}

\begingroup\small\itshape
The aggregated result summaries underlying every figure and table in this article---the
per-configuration outcome summaries, the pairwise statistical comparisons, and the multi-seed
replication checks---are provided with this article as Supplementary Information. All results are
generated by the evaluation simulator under a fixed, content-hashed calibration and the recorded
random seeds described in the Methods; together with the figure and table generators they regenerate
every reported number. The complete raw per-trial outputs, together with the implementation, are
openly available in a single Zenodo record (DOI: 10.5281/zenodo.20932989,
https://doi.org/10.5281/zenodo.20932989) and in the public GitHub repository
https://github.com/helghareeb/ns-dsl. The data are released under the Creative Commons Attribution
4.0 International licence and the bundled software under the MIT licence.
\endgroup

\medskip
We believe this fully addresses the comment, and we thank you for your time.
